一、项目介绍

基于 Spring Boot + Vue 的茶叶电商系统，系统角色包括 超级管理员（后台） / 系统管理员（运营） / 供应商/采购 / 普通用户（前台），主要功能如下：

超级管理员（后台）：
基本操作：登录、修改密码、获取/修改个人信息
用户管理：获取用户列表、查看/修改/删除用户、会员等级与角色分配
角色与权限管理：新增/删除/查询角色、权限分配
商品管理：分类/品牌/规格管理、商品增删改查、上下架与库存管理
订单管理：查询订单、确认发货、处理退款/退货、查看物流
采购与供应商：供应商管理、采购单管理、入库/库存调整
运营管理：轮播图（Banner）、活动/促销管理
统计与报表：后台概览/仪表盘（ECharts 展示）、各类统计接口
文件与媒体管理：图片/文件上传（阿里云 OSS）、文件删除与管理
系统服务：短信与邮件设置、支付配置（支付宝）、API 文档（Knife4j）、日志与监控

系统管理员 / 运营：
基本操作：登录、个人信息管理
运营相关：管理 Banner、活动、查看统计报表
订单与用户：协助处理订单、查看/管理用户（权限与操作受限于角色）

供应商 / 采购角色：
供应商信息查看与维护
创建/管理采购单、查看采购记录、与商品入库相关操作

普通用户（前台）：
用户功能：注册、登录、个人信息与会员中心
商品浏览：按分类/品牌/关键词查询商品、查看商品详情
购物流程：加入购物车、下单、支付宝支付集成
订单管理：查询订单、查看物流、申请退货/退款
文件（头像等）上传（通过后端 OSS 接口）、接收短信/邮件通知

二、项目技术
编程语言：Java（后端），JavaScript (ES5/ES6) + Vue 2.x（前端）
项目架构：B/S（前后端分离）
前端技术：，Vue 2.x、vue-router、vuex，Element UI，axios，echarts / v-charts
后端技术：，Spring Boot 2.x、Spring 生态，MyBatis-Plus、Spring Data JPA，MySQL，Redis，，阿里云 OSS SDK，alipay-sdk（支付宝）、阿里云短信 SDK、邮件（Spring Mail）

三、运行环境
JDK版本：1.8及以上都可以
操作系统：Windows7/10、MacOS
开发工具：IDEA、Ecplise、MyEclipse都可以
数据库: MySQL5.5/5.7/8.0版本都可以
npm版本：6.14.13及以上都可以
Redis版本：3.2.100及以上都可以

四、数据库配置文件
文件名：application.yml、application-jdbc.yml、application-mail.yml、application-redis.yml
编码类型：utf8
